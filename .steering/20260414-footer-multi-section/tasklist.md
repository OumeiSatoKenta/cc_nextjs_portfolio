### 学んだこと

- Biome の nursery ルール `useSortedClasses` により CSS クラスの記述順序が自動整列される（`border-gray-100 border-t` の順）。
- `within()` + `aria-label` によるスコープ絞り込みは、DOM インデックス依存のクエリより堅牢でアクセシビリティにも貢献する。
- Server Component では `new Date()` がビルド時に評価される点を認識。Static Export 運用では年次ビルドが必要（現時点では実用上問題なし）。
