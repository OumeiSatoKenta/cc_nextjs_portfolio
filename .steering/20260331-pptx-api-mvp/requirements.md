# 要求定義: PPTX生成API MVP

## 背景・目的

外部サービスの `pptx-api-506729517706.asia-northeast1.run.app` に依存した `pptx-build` スキルを、
自社運用のAPIに切り替えるためのMVP実装。

セキュリティ上の懸念（機密データの外部送信）を排除し、社内クローズドな環境でPPTX生成を完結させる。

## スコープ（MVP: 5テンプレート）

### 実装するエンドポイント

1. `GET /templates` — テンプレート一覧＋ゾーン制約を返す
2. `POST /build` — スライドJSONを受け取りPPTXバイナリを返す

### 実装するテンプレート（5枚）

アジコイネの全フローをカバーする最小セット：

| ID | 役割 | アジコイネ |
|----|------|-----------|
| T81 | 表紙（センター型） | - |
| T59 | 課題・実情とありたい姿のギャップ | ア＋ジ |
| T62 | サービスの機能一覧（3機能） | コ |
| T66 | 導入事例（インタビュー形式） | イ |
| T72 | ネクストステップ | ネ |

### 対象外（MVP外）

- 認証・APIキー管理（ローカル実行前提）
- グラフテンプレート（T74〜T80）
- 全106テンプレートの実装

## 技術要件

- **言語**: Python 3.x
- **フレームワーク**: FastAPI
- **PPTXライブラリ**: python-pptx
- **実行環境**: ローカル（localhost）
- **インターフェース**: pptx-buildスキルの既存JSONスキーマと互換

## JSONスキーマ（/buildのリクエスト形式）

```json
{
  "slides": [
    {
      "template_id": "T81",
      "title": "スライドタイトル",
      "key_message": "キーメッセージ",
      "zones": {
        "zone1": "テキスト内容",
        "zone2": "テキスト内容"
      }
    }
  ]
}
```

## 成功基準

- `GET /templates` が5テンプレートのゾーン制約を返す
- `POST /build` に上記JSONを渡すとPPTXファイルが返ってくる
- 生成されたPPTXをPowerPointで開いて内容が表示される
- pptx-buildスキルのSTEP 8がこのAPIで動作する
